## **What you can do with it?**

You can ask [Dume.ai](http://Dume.ai) to go through your open issues and provide suggestion, or review PRs based on your behalf.

![Github Demo](/images/github_demo.png)

## How to integrate?

1. Login to [Dume.ai](http://Dume.ai)
2. Go to [Integrations](https://app.dume.ai/integrations) and select available integrations.
3. Click connect, you'll be redirected to Github and allow the permissions.
4. Once OAuth is complete, [Dume.ai](http://Dume.ai) can then access github repos.
5. You can tell Dume fetch open PRs, open Issues, review PRs.

## Example commands

- Fertch all my the PRs assigned to me and review them for security vulnerablitites.
- Go through this open issue and summarise all the comments and chekc if a similiar duplicate issue exist.
- Fetch all open PRs from this repo and give a summary of all the changes.
