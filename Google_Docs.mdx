## **What you can do with it?**

You can ask [Dume.ai](http://Dume.ai) to create summarise or search for a google doc. It can also create and new doc file and save to it.

## How to integrate?

1. Login to [Dume.ai](http://Dume.ai)
2. Go to [Integrations](https://app.dume.ai/integrations) and select available integrations.
3. Click connect, you'll be redirected to Google Docs and allow the permissions.
4. Once OAuth is complete, [Dume.ai](http://Dume.ai) can then access all your Google Docs.
5. You can tell Dume read, create, and search google docs on your behalf.

## Example commands

- Search me the PRD for this project in Google Docs.
- Can you save the summary of our conversation to google docs.
- Please fetch all the docs I had updated yesterday.
